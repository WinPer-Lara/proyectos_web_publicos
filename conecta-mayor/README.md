# Conecta Mayor — Sitio de aprendizaje para personas mayores

Proyecto de ejemplo: estructura inicial con HTML, CSS y JavaScript separados.

Instrucciones rápidas:

1. Abrir `index.html` en un navegador moderno (doble clic o "Abrir con" desde tu explorador de archivos).
2. Opcional: servir con un servidor local si usas módulos o evitar problemas de CORS.

Estructura principal:
- `index.html` — página de inicio (wireframe).
- `pages/` — páginas de módulos (ej. `computacion-basica.html`).
- `css/styles.css` — hoja de estilos principal con variables accesibles.
- `js/scripts.js` — controles de accesibilidad (tamaño de fuente, alto contraste).

Simuladores interactivos:
- `js/simulators.js` contiene simuladores para correo, chat, videollamada, seguridad y telemedicina.
- Abre las páginas en `pages/` y prueba los simuladores: redactar correo, enviar chat, unirse a reunión, checklist de seguridad, agendar cita.

Nota sobre recursos visuales:
- Las imágenes de ejemplo usan Unsplash (enlaces externos) para miniaturas y portada.
- El vídeo de ejemplo está enlazado a un vídeo público de muestra (Big Buck Bunny) solamente como placeholder; reemplázalo por tus vídeos instructivos reales cuando los tengas.

Recomendaciones:
- Probar con usuarios reales y ajustar tamaños/contrastes según feedback.
- Sustituir vídeos y recursos por contenidos locales/propios.
